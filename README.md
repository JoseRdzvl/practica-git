# practica-git

## Tipo de Servicio
el servicio sigue las reglas REST (por simplicidad, facilidad y flexibilidad).

Aparte de las 3 cosas que se mencionan lo elegi por lo siguiente:
- REST permite intercambio de datos ligero usando JSON.  
- Es más fácil de consumir desde distintos clientes (web, móvil).  
- Mejora el rendimiento y escalabilidad frente a SOAP.
